Follow
====================

Normal Path
********************
	User					Program	
1.						Menampilkan halaman utama
2.	Melakukan Pencarian user	
3.	Klik tombol Follow
4.						Update database
Selesai

Alternative paths
********************
User melakukan Pencarian User melalui fitur filter pada langkah 2
2.	Memilih kriteria yang akan difilter
3.						Menampilkan data hasil filter
4.	Lanjutkan normal path pada langkah 3
Selesai.

User melakukan Pencarian user melalui kolom search pada langkah 2
2.	Input data di kolom search
3.						Menampilkan data hasil filter
4.	Lanjutkan normal path pada langkah 3
Selesai.

User menekan tombol Unfollow pada langkah 3
3.	Menekan tombol Unfollow		
4.						alert konfirmasi
5.	Menekan tombol konfirmasi
6.	Lanjutkan normal path pada langkah 4
selesai.

Respon
********************
Tombol follow di ganti menjadi Unfollow, setelah user telah follow user lain
Tombol Unfollow menjadi Follow, setelah user menekan tombol unfollow user lain
